An example demonstrating how long and messy code of nested callbacks can be made simple and adorable with promises and even better with async and await.

The task here is to move the buttons towards right by some pixels with a time delay multiple times until it can not move further. The same code is implemented in three ways. for first button it is implemented with nested callbacks. for second button it is implemented with promises. for third button its is implemented with async and await. You can adjust the window size to see the difference in execution.
